🔒
Your keys are stored only on this device
API keys are saved in your browser's localStorage only. They are never sent anywhere except directly to Calendly, Typeform, or ActiveCampaign when you click Sync. Treat them like passwords.
📅
Calendly
Not connected
Pulls upcoming bookings into Events. New bookings auto-create Prospect clients.

Calendly → Account Settings → Integrations → API & Webhooks → Generate New Token

📋
Typeform
Not connected
Pulls form submissions and creates new Prospect clients with answers saved as notes.

Typeform → Profile photo → Settings → Personal tokens → Generate a new token

📨
ActiveCampaign
Not connected
Syncs your ActiveCampaign contacts into the Clients list as Prospects.

ActiveCampaign → Settings (gear icon) → Developer → copy both API URL and API Key
